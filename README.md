## CREATED AN EVIRONMENT
```
1. TO CREATE AN ENVIRONMENT :- conda -p Nenv python=3.9(the python version you have)
2. TO LOAD REQUIREMNETS (All necessary packages) FILE :- pip install -r requirements.txt
3. To make the folder into packages we have to create setup.py
4. If you to execute the setup.py individually then use the command- python setup.py install
```
